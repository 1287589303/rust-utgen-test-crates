self.pattern: Any string containing a backslash followed by characters meeting various conditions; tests include sequences starting with '\\d', '\\s', '\\w', '\\D', '\\S', '\\W', '\\p', '\\P', '\\x', '\\u', '\\U', and '\\r'; the overall pattern must not contain unsupported escape sequences or characters following the backslash that are considered meta or escapeable.
