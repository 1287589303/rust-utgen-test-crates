self.bits should be set such that Look::WordStartHalfUnicode (bit 16) is included and all other bits for Look::WordUnicode, Look::WordUnicodeNegate, Look::WordStartUnicode, Look::WordEndUnicode, Look::WordEndHalfUnicode should be unset (0). Valid range of self.bits is 0x00010000 (for WordStartHalfUnicode only) to 0x00010000.
